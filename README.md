# wp1.7-slc9-gpu-builder-image
Docker image to run the [Alice O2](https://github.com/cern-nextgen/AliceO2) standalone benchmark with C++26 reflections.

## Pull the generated image
The github workflow [wp1.7-slc9-gpu-builder:latest](https://github.com/cern-nextgen/wp1.7-slc9-gpu-builder-image/actions/workflows/docker-image.yml) will build and push an image that can be pulled as follows.
```
docker pull registry.cern.ch/ngt-wp1.7/wp1.7-slc9-gpu-builder:latest
```
